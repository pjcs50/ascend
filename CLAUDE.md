# Personal OS

This is a single-user, private life-management web app (habits, people, journaling, personal growth) synced across phone, laptop, and iPad.

**The full build specification lives in [personal-os-v1-spec.md](personal-os-v1-spec.md). Read it and follow it.** It is the working agreement for this project — tech stack, navigation, data model, and build order are all defined there.
